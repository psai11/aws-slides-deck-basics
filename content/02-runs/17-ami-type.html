AMI & instance type 17
Boot image & box

An AMI is the boot image. The instance type is the box.

naming Type names: a family letter (t/m/c/r) + a generation (3, 5, 6i, 7g) + a size (.micro, .large, .4xlarge).
AMI = starting state of the disk. Instance type = the hardware shape. Both chosen at launch; both changeable later.
