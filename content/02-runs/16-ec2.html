EC2 16
Elastic Compute Cloud

EC2 is a virtual machine. You operate it.

EC2 = a VM in a datacenter.
  • ·  You pick an AMI (boot image) and an instance type (CPU + memory).
  • ·  Full guest-OS control: install, run long-lived processes, open ports.
  • ·  You manage patching, scaling, and pay for capacity while it runs.
Pricing models —
On-Demand · Savings Plans · Reserved · Spot.
EC2 is the building block. Everything else in compute is a higher abstraction. If you want a server you SSH into, this is it.
