Three flavours 15
Compute

Three flavours of "somewhere your code runs."

VMs

EC2 — you pick OS, size, run anything you want, manage the box.

Functions

Lambda — run code on events, no servers to patch.

Containers

ECS / EKS / Fargate — package once, schedule many.

You'll combine them in real systems.

Don't make this either/or. Real systems mix all three. Today's slide is the vocabulary.
