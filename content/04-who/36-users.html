IAM users

An IAM user is a named identity.

  • ·  Unique name in your account.
  • ·  Console password (with MFA) and/or access keys for CLI/SDK.
  • ·  Access keys are long-lived secrets — leak = blast radius.
rule Prefer roles over IAM users for applications. Always.
IAM users: starting point and wrong default for apps. Users for humans, sparingly; roles for everything else.
