Anatomy

Effect · Action · Resource · Condition.

{
  "Effect":   "Allow",
  "Action":   "s3:GetObject",
  "Resource": "arn:aws:s3:::acme-prod-data/reports/*",
  "Condition": {
    "IpAddress": { "aws:SourceIp": "203.0.113.0/24" }
  }
}
Effect — Allow or Deny. Deny wins ties.
Action — the API call. Wildcards permitted (use sparingly).
Resource — the ARN(s) the action targets.
Condition — the safety net (here: only from the corp IP).

Read it as: role X may read /reports/* from acme-prod-data, but only from the corp IP range.

The slide where the abstraction becomes concrete. Read the JSON aloud and translate each pillar into English. The Condition is the bit beginners miss — it's the safety net.
