Elastic File System

EFS is a network drive many EC2s share.

  • ·  Shared access. Many readers/writers, one filesystem.
  • ·  Regional durability by default (One Zone option for cheaper non-critical).
  • ·  Good for content farms, web tiers, lift-and-shift NFS apps.
EBS = one volume, one instance. EFS = one folder, many instances.
This slide disambiguates EBS from EFS. Same word "storage," different shape. EFS when multiple EC2s need the same files.
