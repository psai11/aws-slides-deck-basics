Storage shapes

Three storage shapes.

object

S3

Buckets and keys. HTTP API. Internet-scale blobs.

block

EBS

Virtual disk. Attached to one EC2 in one AZ.

file

EFS

Shared NFS folder. Many EC2s mount it together.

— plus instance store: fast, ephemeral, host-local — for scratch only.

Three shapes, three access models. Object = HTTP. Block = mounted disk. File = network share.
